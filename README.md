# 🌱 About the Project

**OptiCrop** is an intelligent agriculture support system that helps farmers make better crop decisions using **Machine Learning**. The application analyzes environmental and soil parameters to recommend suitable crops, helping improve productivity and sustainable farming.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/238x1a0412-krishna/opticrop_project.git
```

Go to project folder

```bash
cd opticrop_project
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python app.py
```

---

# 🚀 Usage

1. Open the application.
2. Enter the required soil and environmental values.
3. Click **Predict**.
4. View the recommended crop.
5. Use the recommendation for better agricultural planning.
